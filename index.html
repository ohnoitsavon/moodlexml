Moodle XML Helper

"Q QUESTION TYPE"
Sets the type of question.
q multichoice|shortanswer|truefalse|numerical

"N QUESTION NAME"
Sets the question name.
n Question 1

"T QUESTION TEXT"
Sets the question text.
t What colour is the sky?

"A = ANSWER"
Sets the correct answer.
a = Blue

"A ~ ANSWER"
Sets the incorrect answer.
a ~ Red

"A 50 ANSWER"
Sets an answer with a percentage grade, used with multi-answer questions.
a 50 White

"FB FEEDBACK"
Used after an answer to provide feedback.
fb Well Done

"GFB GENERAL FEEDBACK"
Used to provide general feedback.
gfb This was an easy question

"END"
Always put this at the end of a question.
end

Multiple Choice

"SINGLE TRUE/FALSE"
Toggles between allowing single or multiple answers.
single true|false

"SHUFFLE TRUE/FALSE"
Turns shuffle within question on or off.
shuffle true|false

"SELECT NUMBERS/LETTERS"
Sets the answer numbers/letters.
select abc|ABCD|123|iii|IIII|none

True/False

"TRUE"
Sets true as the correct answer.
true

"FALSE"
Sets false as the correct answer.
false

Simple Example
q multichoice
n Question 1
t What colour is the sky?
a = blue
a ~ red
a ~ green
a ~ yellow
end

Full Example
q multichoice
n Question 2
t Which of these are makes of car?
a 50 Ford
fb Correct!
a 0 Rond
fb Wrong!
a 50 Subaru
fb Correct!
a 0 Asda
fb Wrong!
single false
shuffle true
select 123
gfb This was an easy question
end
